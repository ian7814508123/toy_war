# 玩具戰爭：基地操作指南

本手冊說明目前版本已實作的基地經營功能與操作流程。

## 1. 基礎操作
*   **鏡頭移動**：使用鍵盤 `W` `A` `S` `D` 或 **按住滑鼠中鍵** 拖曳地圖。
*   **縮放**：使用鍵盤 `Q` (放大)、`E` (縮小) 或 **滑鼠滾輪**。
*   **選取建築**：左鍵點擊地圖上的建築或右側「建築清單」中的卡片。
*   **取消選取**：右鍵點擊地圖空白處或點擊「清除選取」按鈕。

## 2. 建築與升級
*   **建造**：在清單選取建築後，在地圖格線內點擊左鍵放置。
    *   綠色預覽代表可放置。
    *   紅色預覽代表重疊、超界或超過數量上限。
*   **拖曳移動**：按住已建造的建築不放，可直接拖曳至新位置。
*   **升級**：選取建築後，若資源足夠且滿足主控台等級限制，可點擊「升級」按鈕。
    *   **限制**：所有建築的等級上限受限於目前的「主控台」等級。

## 3. 資源管理
*   **自動生產**：資源工廠會根據等級持續生產資源至「暫存上限」。
*   **收取資源**：當建築變色（橘色代表滿載）時，選取建築並點擊「收取」以轉入基地總庫存。
*   **儲存上限**：總庫存上限由「倉庫」的總數與等級決定。若庫存已滿，工廠將停止生產。

## 4. 士兵訓練
*   **生產士兵**：選取「一般士兵工廠」或「特種士兵工廠」後，右側會解鎖可訓練的兵種按鈕。
*   **佇列系統**：點擊兵種按鈕後會進入該建築的訓練佇列。每個建築的佇列與進度都是獨立的。
*   **查看庫存**：點擊主選單的「部隊」分頁，可查看基地內目前所有擁有的兵力統計。

## 5. 存檔
*   本遊戲具備 **自動存檔** 功能（每 5 秒一次）。
*   您可以透過頂部的「雲端存檔」區塊，輸入 Email 使用雲端同步功能。
