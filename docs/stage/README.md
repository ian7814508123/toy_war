## 目前積壓的想法 (Backlog)

### 1. 英雄系統 (Phase 2)
*   **領養機制**：在英雄大廳等級提升後，解鎖更多英雄（上限 6 位）。
*   **餵食與等級**：實作士兵餵食英雄以提升經驗值的邏輯。
*   **法力與技能**：英雄參與攻擊消耗 80 法力，參與防守消耗 15 法力。法力隨時間自動恢復。

### 2. 戰鬥系統 (Phase 3)
*   **場景切換**：新增 `CombatScene` 負責進攻模擬。
    * 設計 **進攻模式** 關卡: 玩家進攻敵方基地(敵方由我們設計)
    * 設計 **防守模式** 關卡: 玩家防守敵方進攻(敵方由我們設計)
*   **兵力投放**：玩家在基地外圍點擊以投放士兵。
*   **自動戰鬥 AI**：
    *   士兵依據 `targetPriority`（如 `defense_first`）尋找最近目標。
    *   防禦塔自動掃描射程內敵人並攻擊。
*   **掠奪計算**：士兵摧毀資源建築後，按比例（及士兵特性，如蜘蛛加成）增加戰利品。

### 3. 美術資源替換
*   目前的場景使用幾何圖形（Phaser Graphics）渲染。
*   未來應根據 `遊戲元素/` PDF 中的截圖進行像素畫切圖（Aseprite），並在 `PreloadScene` 載入以替換目前的色塊。(不排除使用建模軟體製作各式建築)

### 4. 土地擴張
*   實作「土地擴張」建築，增加可放置面積。

### 5. 清單分頁
*   目前的清單同時包含建造、建築升級、士兵訓練、資源收取等功能,佔據畫面右側過長的篇幅。
*   未來應將清單分頁，每個分頁包含相關功能,當點擊該類建築,該類建築的清單才會顯示出來。
